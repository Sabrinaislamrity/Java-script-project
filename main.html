Question-1: What are the different ways to select an element in the DOM?

Here are the different ways to select an element in the DOM:
document.getElementById("myID");
getElementById()
getElementsByClassName()
getElementsByTagName()
querySelector()
querySelectorAll()
closest()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML is Gets/sets HTML content, including tags.
innerText is Gets/sets only visible text, ignores hidden elements.
textContent is Gets/sets all text, including hidden ones.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where a parent element handles events for its child elements, even if they are added later.

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event starts from the target element and moves up to its parent elements in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

for Create an Element we can use :
let newDiv = document.createElement("div"); newDiv.textContent = "Hello";
for Add (Append) the Element we can use :
document.body.appendChild(newDiv);
for Remove the Element we can use :
newDiv.remove();
